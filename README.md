# kaffee-web

## Deploy :rocket:

Pushing to master will deploy with Netlify. The website
can be deployed locally by building and starting the
docker image.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

### Run your unit tests

```
yarn run test:unit
```
